| ATmega328P Pin | Function               |
| -------------- | ---------------------- |
| PC4 (A4)       | OLED SDA / MPU6050 SDA |
| PC5 (A5)       | OLED SCL / MPU6050 SCL |
| PD3            | Buzzer                 |
| PD2            | Push Button            |
| PB5            | Status LED             |
| VCC            | +5V                    |
| GND            | Ground                 |
| RESET          | ISP Programming        |
| XTAL1          | 16 MHz Crystal         |
| XTAL2          | 16 MHz Crystal         |
